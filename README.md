成绩管理系统（Node.js 版）
轻量后端系统，基于 Node.js + Express + MySQL 构建，支持多角色权限管控，可灵活部署在本地、Render 云平台或自有服务器。
核心功能
多角色登录：管理员 / 教师 / 学生权限分离
基础管理：学生信息增查、成绩录入与查询
数据统计：成绩排名（级部 / 班级）、历史成绩追踪
安全保障：JWT 身份认证、防暴力登录、密码加密存储
跨端部署：兼容本地、Render、自有服务器部署环境
本地快速启动
前置条件
需安装 Node.js 20.x LTS、MySQL 8.0+、npm/yarn
操作流程
克隆代码至本地并进入项目 api 目录
执行命令安装项目依赖
在 api 目录创建 .env 文件，配置数据库地址、账号、密码等基础信息
手动创建 MySQL 数据库，执行初始化 SQL 导入测试账号
启动服务，访问健康检查接口验证启动成功
部署方式
Render 部署
代码推送至 GitHub 仓库
Render 后台导入仓库，配置项目目录、构建 / 启动命令
配置环境变量（数据库改用 Supabase 远程库）
等待部署完成，通过 Render 分配域名访问
自有服务器部署
安装 Node.js、MySQL、PM2 基础依赖
上传代码至服务器指定目录
配置 .env 文件（数据库用服务器本地库）
安装依赖后用 PM2 启动服务（设置开机自启）
可选：配置 Nginx 反向代理优化访问
技术栈
后端框架：Express
数据库：MySQL
认证：JWT
加密：bcryptjs
进程管理：PM2（服务器部署）
跨域：cors
注意事项
本地运行需确保 MySQL 服务已启动，环境变量配置正确
密码哈希值需生成本地兼容版本
公网部署建议配置 HTTPS
Render 免费版闲置 15 分钟休眠，可定时访问接口保活
防暴力登录默认开启，连续 5 次失败锁定 5 分钟
